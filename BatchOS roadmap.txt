-Se souvenir :

Les programmes BatchOS (.bexe) sont des scripts Batch compilés puis renommés. Ils peuvent utiliser toutes les commandes DOS et avancées,
mais pas de goto ni de transfert de variable vers le script principal (pour l'instant). Ils recoivent toutes les variables du script
principal (Sauf rootPass par mesure de sécurité)

Pour programme repo/FTP : Il faut lancer ftp -s:FILENAME ou filename est un script de commandes FTP à exécuter.

FTP BatchOS.run :
batchos.run

Codes d'erreur Batch Package Manager (BPM) :
BPM-NOLATESTBAT ------------------------------- Le fichier upgrade/latest.bat n'a pas été trouvé après l'exécution du script FTP
BPM-NOUPGRADEBAT ------------------------------ Le fichier upgrade/%VER%/upgrade.bat n'a pas été trouvé après le script FTP
BPM-NOBEXE ------------------------------------ Le fichier .bexe du programme à désinstaller n'a pas été trouvé.
